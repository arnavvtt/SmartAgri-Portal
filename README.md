A Django-based web application designed to assist farmers in making better
decisions by combining real-time weather information, crop management, and
mandi (market) price data on a single platform.

---

## Problem Statement

Farmers often face unpredictable weather conditions and lack transparency
in crop market prices, which leads to poor planning and reduced profits.
There is a need for a unified system that provides reliable weather insights,
crop-based planning support, and accessible mandi prices.

---

## Proposed Solution

Smart Agriculture Portal provides:
- Farmer authentication and profile-based access
- Crop logging for personalized insights
- Weather-based advisories connected to crops
- Government mandi price data for better selling decisions

The system converts raw data into understandable insights to support
day-to-day farming decisions.

---

## Key Features

- User registration and secure login
- Crop management (add, edit, delete crops)
- Farmer-centric weather insights using OpenWeather API
- Live mandi price data using Government of India APMC API
- Dashboard with integrated farm planning support

---

## Technologies Used

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript (Templates)
- **Database:** SQLite
- **APIs Used:**
  - OpenWeather API (Weather data)
  - data.gov.in APMC API (Mandi prices)
